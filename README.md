# one-key-emea-milwaukee-tool
Full stack project as part of the application for a working student position with the ONE-KEY EMEA Milwaukee® Digital Product team (Weekend Challenge)

# Context
A technology company is developing new products and regularly sends physical sample units to different
teams across several European countries so they can be tested and demonstrated before launch.
Because these samples move between countries and teams, it’s difficult to maintain a clear overview of
where each item currently is or what its status might be.
Your challenge is to build a simple system that solves this problem: a central tool list that can assign items
to different country accounts and keep everything synchronized.
